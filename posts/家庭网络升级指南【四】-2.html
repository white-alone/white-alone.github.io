本页目录

时隔3年，家里添了三宝，不管是工作还是生活都有了巨大的变化，最近用了一台相当有意思的NAS产品，于是就有了《家庭网络升级指南》系列的第四篇，给大家介绍下这个很有工程师精神的产品：懒猫微服。

0x10 缘起 ​

懒猫刚发布时候就有看到，不过当时没有太在意，NAS么，我用了15年了，能有啥新花样？AI NAS，一般也就RK方案，搞下NPU、TPU玩点人脸识别这种，不算什么。intel？那玩意是能跑AI的？不得外挂个计算棒什么的？而且NAS这么多年了，群晖、威联通、极空间、绿联各种硬件，软件的话Unraid、TureNAS、FreeNas、OMV、IstoreNAS等等等，早就铺天盖地了，所以，看过就过了，没有啥兴趣和动力去折腾，不过，这外观真好看！

再后面，在群里就认识了CEO王总，一个很有意思的技术人，也是emacs的那位懒猫，居然还是deepin linux的联合创始人，这就有点兴趣了，技术人的产品，大多符合技术人的审美和需求~~

PS：虽然用了HHKB十多年，我依然是坚定地VIM党，但这不影响我对开源大佬的景仰~~

然后就到24年12月了，哥们凌御在群里说他朋友公司准备搞点礼品年会抽奖，问极空间的NAS适不适合给工程师，工程师？那不得看看懒猫？于是就群里就说要不看下懒猫微服？我有他CEO的VX，就这么一来二去，和王总聊起来了。虽然哥们的朋友最后选了极空间，但哥们自己买了个，然后用了说还不错，CEO人很赞，CTO亲自改代码推送，支持非常Nice~~

PS：极空间的创始人还是360出来的，谭总还在朋友圈打过广告，不过有个同事买了，说用的还行，但是呢，论坛里吐槽下就被封号了，这让觉得有点霸道，所以这么多年，我是没真正用过极空间的，只看过评测，但也有很多同事是极粉。

再后来，因为公司的某人的原因，我也准备自己搞点东西了，于是就和哥们一起搞了个SDR产品SignalSDR Pro在CS上众筹，在预热阶段找王总帮忙转发下，他二话不说立马转了，再后面硬件方面讨论多了熟悉了，觉得懒猫的硬件应该不错，毕竟CEO这样吹毛求疵的极致追求，于是就下单搞了一台~~ https://www.crowdsupply.com/signalens/signalsdr-pro

0x20 外观 ​

黑白2种外壳，CNC工艺，非常有质感，很有机甲的感觉，非常小巧精致，可以一手掌握，完全不像一个放机柜里的NAS，像一个摆件，摆在电脑旁、电视旁毫无违和感，而懒猫本身也可以当电视盒用，HDMI直接输出到电视看视频照片等等。

PS：懒猫的外壳成本可能是市面上消费类NAS中最高的，CNC加工阳极氧化，这质感和铁皮钣金差十万八千里，就是成本会爆炸~~

这个产品的定义就不是传统NAS，在后面使用中也确实是这样，使用体验很奇特，根本不像一个NAS，难怪叫懒猫微服，而不是懒猫NAS~~

0x30 硬件配置 ​

懒猫的硬件配置在NAS产品中算高的，Intel Core i5-1155G7 处理器，四核八线程，2.5GHz 主频，睿频4.5GHz，支持 AVX-512 AI推理加速指令集，支持 KVM 硬件虚拟化加速指令集，支持 OpenVINO/IPEX-LLM AI推理加速平台，板载1个DDR4 3200笔记本内存插槽最大32内存，1个PCIe 4.0 x4 m2 2280接口可以上8T的SSD或者改U2接口上U2盘，2个sata 2.5接口可以上机械或者sata ssd，板载2.5G网卡，Intel WiFi 6E AX210+蓝牙5.3，3个USB 3.0，2个TypeC 3.2，1个3.5耳机孔，1个HDMI 2.1 4K口。

主体使用2块PCB叠层组合，这样才能做到足够的外设和接口，拆几个螺丝就能抽出主体，可以很方便的更换外壳，懒猫的外壳是可以单独购买的，当然，如果运气够好，也能抽奖抽到~~

i5-1155G7虽然比不上我的i7-1185G7软路由、5825U还有7840HS的小主机，但和那些RK3568/J4125/N97/N100比，不是一个等级的，有的朋友可能觉得有点贵，可再看看着小巧的体积和这质感一流的CNC外壳，一个好的手办可能都比这贵了。。。

PS：sata盘直接上ssd，做全闪NAS安静得很，体积115x115x115mm的NAS，m2x1+satax2，每块盘8T，24T应该足够应对大多数个人用户以及小企业，唯一的瓶颈可能就是2.5G网口了，用C口可以上5G网卡，不过对于绝大多数用户，2.5G是够用的。

另外，懒猫后续还有备份仓、算力仓等外设，用来拓展容量和算力，作为家庭和小企业的服务中心应该是OK的，所以叫“懒猫微服”。

0x40 系统和应用 ​

懒猫的底层系统LZCOS是基于debian开发的，上面再封了系统和应用，官网也有非常非常多的攻略教程。懒猫深层次的技术原理这里就不展开讲了，我就简单说几个懒猫上我觉得非常实用的功能。

0x41 内网穿透&异地组网 ​

这个是我觉得最实用的功能，群晖的QuickConnect卡不卡？黑群能不能用？Qnap的myQNAPcloud自从在大陆建服后是快了不少，但够快么？绿联的就不说了，极空间没用过不评价，自己折腾的话就是frp、nps等，或者zerotier、wireguard、tailscale等sd-wan工具，为了能随时随地访问nas里的数据，难倒了多少小白用户？所以极空间、绿联一开始能吸引很多用户，因为操作够简单够方便。

懒猫的手机APP有网络模式可以选择，有没有眼熟？用zt、wg、ts那些app是不是也这样？没错，懒猫的客户端就可相当于sd-wan的节点，只要运行了客户端，就相当于进了你的懒猫小专网，所有客户端之间可以互访，再配合局域网转发工具，异地组网简直不要更简单！

懒猫自己开发了一套类似zt、ts的应用，可以做到无公网IP时，NAT3 100%的穿透率，NAT4 25%的穿透概率，可以实现没有公网IP，在外手机和电脑就可以访问家里的服务器和云服务，而且自带2FA，体验类似苹果的二次验证，如果无法穿透，他家的cdn也很舍得花钱，带宽很高，比自己99的阿里云/腾讯云中继肯定强太多了。

比如说，你在杭州放一个懒猫微服，里面存放的资料，不需要折腾什么frp、zt，不需要搞服务器做中继，也不需要各种改配置，仅仅需要你在app上给北京的朋友开一个账号，设置好权限，他装上懒猫客户端，登录账号就能在电脑、手机、平板上看里面的资料，像icloud、onedriver一样直接上传或者下载资料文件，方便么？在老家的父母想看看娃的照片视频，想看电视电影，给他们装好app，打开就能用。

再比如程序员撸代码，自建代码仓库，随时随地，不喜欢gogs？没事，应用商店1000+app，总有一款适合你，你也可以自己移植APP，老王一款给100红包，一个月移植几个，用不了多久懒猫就免费了 ~~

懒猫的开发者中心，就是给各位大佬们准备的，原创应用、移植应用，排行榜前五的大佬们，红包收入都过万了，我把自己常用的一些应用移植上去，得到奖励也是美滋滋的，感谢老王 ~~ 哈哈哈

0x42 AI相册&自然语言搜图 ​

这也是懒猫的AI场景最直观体现，自然语言搜图效果是目前用过的所有相册中最好的。

说起AI相册，最开始用的是群晖Synology的Moments，后面成了Photos，用人脸识别整理相册，一开始很惊艳，为这个还研究怎么开黑群的硬解、新硬件怎么打patch用cpu软解，还有qnap的QuMagie、开源的PhotoPrism、immich还有mt-photos，以及飞牛OS的相册，都做的不错，但这些相册软件，要么硬件太弱，要么要外挂显卡，而且在自然语言处理方面，只能说能用，离好用还比较远。这些里面，我对比后，最终选择的是MT-Photos，功能很强，终身版也才99，不过安装和部署，对于不懂的小白而言，还是有一点点门槛的，特别是配置AI加速和地理位置API这一块。

而懒猫相册，开箱即用，手机电脑pad上随开随用，体验很有icloud的感觉，手机也能很方便的上传和备份。在积累了多年的照片后，每次要找照片会不会抓狂？自然语言搜图，能非常方便的帮你找你需要的照片。

0x43 游戏 ​

懒猫微服，毕竟带个“服”字，所以开游戏服肯定是基操，应用市场有非常多爱好者移植的各种游戏，也有各种平台，比如传奇、帕鲁、饥荒、我的世界、异星工厂、雾锁王国、雨中冒险2等等，后续也还会有更多的游戏上架，几个同学好友依托懒猫的网络，联机打把红警、帝国时代、dota、cs、星际都不是梦，拥有一台懒猫，就拥有懒猫的小专网。

0x44 其他应用 ​

懒猫清单、通讯录、智慧屏、文字识别这些应用都是基本盘，应用市场一千多应用，还有社区在不断增加新的应用，像群晖、qnap传统nas根本不可能有这么多app，有需要只能自己从三方市场找或者部署docker。 https://lazycat.cloud/appstore/#/shop

0x45 二次开发 ​

懒猫的理念和架构设计都非常工程师，在开放性做的非常好，有文档，有工具，VIP群N对1指导，应用移植、开发上架有奖励，配置也不错，作为开发机跑各种应用代码也足够，微型服务器，名副其实。 https://developer.lazycat.cloud/ 在应用市场有非常非常多的开发工具，等算力仓发布，跑70B模型，无限token随便耍~~

0x50 售后服务 ​

这也是懒猫被很多人吹爆的地方，当你下单了一台懒猫，就会有一个专属VIP服务群，任何问题、需求都会响应，复杂的问题甚至会有CTO亲自出手解决。。。

功能需求？记录下去，排期~~ bug问题？临时解决方案，长期解法~~ 开发问题？直接问，解决不了的等CTO出马就行~~ 意见建议？老王就在群里，直达CEO~~

我用懒猫2个月，都是有问必答，这种VIP服务，群晖？威联通？绿联？极空间？之前买的绿联nas被客服气个半死，在这个群里，你就是VIP~~

另外，懒猫还不定期做活动搞抽奖，多年没中过奖的我都抽中了一个多功能充电器haha

0x60 小结 ​

懒猫微服是个非常优秀的产品，我现在也在逐步将我的各种常用的应用、服务迁移到懒猫上，顺带移植一些应用赚点娃的尿不湿钱。

不过，懒猫毕竟刚面世不到1年，功能特性和传统NAS也有很大不同，有的功能现在还没有，需要依靠应用市场的应用来补充，请给它一点成长的时间，以懒猫200+的开发团队和技术范儿的CEO/CTO调性，未来功能只会越来越强，越来越多，而且懒猫是面向全球售卖的，海外市场客户也有很多，以后会越来越强大的~~

套一个群友的总结：懒猫微服 = 传统NAS + Cloudflare + Tailscale + VPN虚拟专网 + Authy + Nginx + AI大模型 + 微信小程序 + AppStore + KVM/Docker/PVE + iCloud/TimeMachine + Chrome电视

再晒一把我改的懒猫，应该是当前最胖的猫了 ~~
