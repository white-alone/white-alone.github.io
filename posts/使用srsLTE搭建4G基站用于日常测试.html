1гҖҒзЎ¬д»¶е’ҢеөҢе…ҘејҸзі»з»ҹе®үе…ЁпјҡжңүRTOSгҖҒAndroidеҶ…ж ёгҖҒLinuxзі»з»ҹе®үе…Ёз»ҸйӘҢпјӣжңүRISC-Vжһ¶жһ„дёӢзҡ„еөҢе…ҘејҸејҖеҸ‘жҲ–е®үе…Ёз ”з©¶з»ҸйӘҢпјӣзҶҹжӮүеёёи§ҒжҷәиғҪзЎ¬д»¶и®ҫеӨҮ&жүӢжңәзҡ„еӣәд»¶жҸҗеҸ–дёҺеҲҶжһҗж–№жі•пјӣзҶҹжӮүзЎ¬д»¶PCBи®ҫи®ЎгҖҒз”өи·ҜdebugеҲҶжһҗи°ғиҜ•пјӣжңүйҖҶеҗ‘е·ҘзЁӢз»ҸйӘҢпјҢзҶҹз»ғдҪҝз”ЁIDAгҖҒOllyDbgзӯүйҖҶеҗ‘е·Ҙе…·иҝӣиЎҢйқҷжҖҒ&еҠЁжҖҒдәҢиҝӣеҲ¶еҲҶжһҗпјӣ
2гҖҒж— зәҝе®үе…Ёпјҡжңүж— зәҝйҖҡдҝЎејҖеҸ‘и°ғиҜ•е’Ңе®үе…ЁеҲҶжһҗзҡ„з»ҸйӘҢпјӣеҰӮзҶҹжӮүWiFiпјҢ4G/5GпјҢи“қзүҷгҖҒNFCзӯүж— зәҝеҚҸи®®зҡ„е®үе…ЁжјҸжҙһгҖҒж ҮеҮҶи·ҹиҝӣпјӣзҶҹжӮүGNU RadioгҖҒUSRPгҖҒBladeRFзӯүSDRи®ҫеӨҮзҡ„дҪҝз”Ёпјӣжңүж— зәҝеҚҸи®®зұ»жјҸжҙһе’Ңж— зәҝиҠҜзүҮе®һзҺ°зұ»жјҸжҙһжҢ–жҺҳз»ҸйӘҢдјҳе…Ҳпјӣ
3гҖҒдј ж„ҹеҷЁе’ҢиҠҜзүҮе®үе…ЁпјҡзҶҹжӮүдәәи„ёгҖҒжҢҮзә№гҖҒеЈ°зә№гҖҒзңјзә№пјҢжҲ–иҖ…UзӣҫгҖҒеҠ еҜҶиҠҜзүҮзӯүиә«д»ҪиҜҶеҲ«з®—жі•жҲ–е…·еӨҮзӣёе…іејҖеҸ‘и°ғиҜ•з»ҸйӘҢдјҳе…ҲпјӣжңүжҷәиғҪзЎ¬д»¶з ҙи§Јзҡ„зӣёе…іе®һжҲҳз»ҸйӘҢпјҢеңЁGeekPwnгҖҒXPwnгҖҒHackPwnзӯүз ҙи§ЈеӨ§иөӣеұ•зӨәиҝҮз ҙи§ЈйЎ№зӣ®гҖӮ

еҫҲеӨҡIoTи®ҫеӨҮйғҪеңЁдҪҝз”Ё4GзҪ‘еҚЎпјҢдҪҶжөӢ4Gй“ҫи·Ҝе®үе…Ёж—¶еҖҷпјҢжҜ•з«ҹдёҚеғҸWIFIжөӢиҜ•йӮЈд№Ҳж–№дҫҝпјҢCMW500зҡ„д»·ж јиҝҳжҳҜеҫҲиҙөзҡ„пјҢе°Өе…¶жҳҜжғіIoTе®үе…ЁзҲұеҘҪиҖ…з«ҘйһӢпјҢжҲ–иҖ…е…¶д»–з§»еҠЁз«ҜйңҖиҰҒжөӢиҜ•4Gй“ҫи·Ҝзҡ„пјҢе…¶е®һеҸҜд»ҘдҪҝз”ЁдҪҺжҲҗжң¬SDRж–№жЎҲпјҢд№ҹе°ұжҳҜиҝҷйҮҢиҰҒд»Ӣз»Қзҡ„гҖӮ

жңҖдҪҺжҲҗжң¬еҸҜд»ҘдҪҝз”ЁLimeSDR Mini+PCжңәпјҢд№ҹеҸҜд»ҘдҪҝз”ЁBladeRF x40жҲ–иҖ…x115жқҘе®һзҺ°гҖӮеҰӮжһңжқЎд»¶е…Ғи®ёпјҢиҝҳжҳҜдҪҝз”ЁUSRP B200miniгҖҒB210жҲ–иҖ…N210з”ҡиҮіX310жҜ”иҫғиҲ’жңҚгҖӮ

жңҖеҗҺеҶҚж·»еҠ SIMеҚЎзҡ„й…ҚзҪ®пјҢдё»иҰҒжҳҜIMSIгҖҒKIгҖҒOPCпјҢзј–иҫ‘~/.config/srsLTE/user_db.csvж–Үд»¶пјҢж јејҸдёәпјҡ

vim
(ue_name),(algo),(imsi),(K),(OP/OPc_type),(OP/OPc_value),(AMF),(SQN),(QCI),(IP_alloc)
ue3,mil,221010123456789,6874736969202073796D4B2079650A76,opc,504F20634F6320504F50206363500A4F,8000,000000001234,7,dynamic

еҶҚе°ҶиҝҷдәӣеҸӮж•°еҶҷе…ҘSIMеҚЎпјҢиҝҷйҮҢе»әи®®дҪҝз”ЁGPSIMWriteпјҢз®ҖеҚ•дҫҝжҚ·пјҢзӣёжҜ”pysimиҝҳжҳҜж–№дҫҝдёҚе°‘пјҢзјәзӮ№жҳҜеҸӘиғҪеңЁWindowsдёӢгҖӮеҶҷеҚЎеҷЁеҸҜд»ҘдҪҝз”ЁMCR3512гҖҒM100зӯүиҗҘдёҡеҺ…жүҖз”ЁеҶҷеҚЎеҷЁйғҪеҸҜд»ҘпјҢз”ЁBludrive IISIMд№ҹеҸҜд»ҘгҖӮ

зҷҪеҚЎйҖүжӢ©LTEеҚЎе°ұеҸҜд»ҘпјҢеҫҲеӨҡжүӢжңәжҲ–иҖ…з»Ҳз«ҜеҺӮе•ҶйңҖиҰҒз”ЁзҷҪеҚЎй…ҚеҗҲCMW500дёҖзұ»з»јеҗҲжөӢиҜ•д»ӘиҝӣиЎҢжөӢиҜ•пјҢжүҖд»ҘжЈҖзҙўжөӢиҜ•зҷҪеҚЎе…ій”®еӯ—иғҪд№°еҲ°пјҢд»·ж јдёҚдҫҝе®ңгҖӮд»…дҫӣеӯҰд№ дҪҝз”ЁпјҢиҜ·еӢҝз”ЁдәҺйқһжі•з”ЁйҖ”пјҒ

е…·дҪ“еҸҜд»Ҙи®ҫзҪ®е®ҢLTEзҡ„IMSI15гҖҒKIе’ҢOPCд№ӢеҗҺпјҢзӮ№еҮ»Same With LTEпјҢд№ӢеҗҺзӮ№еҮ»Write CardеҶҷе…ҘзҷҪеҚЎпјҡ

жҠҠеҶҷеҘҪзҡ„еҚЎиЈ…е…ҘжүӢжңәпјҢеҶҚеҲҶеҲ«еҗҜеҠЁsrsepcе’Ңsrsenbпјҡ

иҝҷж ·srsLTEе°ұеҗҜеҠЁдәҶпјҢдёӢиЎҢ2685MHzдёҠиЎҢ2565MHzгҖӮеҰӮжһңдёҚдҪҝз”ЁUSRPпјҢдҪҝз”ЁBladerRFжҲ–иҖ…LimeSDRд№ҹзұ»дјјпјҢеңЁеҗҜеҠЁж—¶еҖҷдјҡиҮӘеҠЁеҠ иҪҪгҖӮ

жіЁпјҡеҰӮжһңUSRPгҖҒBladeRFй©ұеҠЁйғҪе®үиЈ…дәҶпјҢдёӘеҲ«дәәзҡ„зҺҜеўғдјҡеҮәзҺ°д»ҺUSRPжҚўBladeRFж— жі•еҠ иҪҪзҡ„й—®йўҳпјҢиҝӣе…ҘsrsLTEзҡ„buildзӣ®еҪ•пјҢжЈҖжҹҘCMakeCache.txtж–Үд»¶дёӯBLADERF_FOUNDжҳҜTRUEгҖӮеҰӮжһңжҳҜFALSEпјҢе°ұеҲ жҺүCMakeCache.txtеҶҚйҮҚж–°вҖңcmake..вҖқпјҢжҢүз…§еүҚйқўзҡ„жӯҘйӘӨйҮҚж–°зј–иҜ‘е®үиЈ…srsLTEгҖӮ

жіЁпјҡдёәдәҶиҺ·еҫ—жңҖдҪіжҖ§иғҪпјҢе»әи®®зҰҒз”ЁCPUйў‘зҺҮеҠЁжҖҒи°ғж•ҙпјҢжҜ”еҰӮд»ҘдёӢи„ҡжң¬пјҡ

shell
for f in /sys/devices/system/cpu/cpu[0-9]*/cpufreq/scaling_governor ; do
  echo performance > $f
done

еңЁsrsenbз•ҢйқўеҸҜд»Ҙиҫ“е…ҘtеӣһиҪҰжқҘе®һж—¶жҹҘзңӢsnrгҖҒblerзӯүжҖ§иғҪжҢҮж ҮеҸӮж•°пјҢеҰӮеӣҫжүҖзӨәпјҡ

еҰӮжһңз”өи„‘й…ҚзҪ®й«ҳзҡ„иҜқпјҢеҸҜд»Ҙжү“ејҖжҳҹеә§еӣҫпјҢдҝ®ж”№enb.confпјҢе°Ҷenableи®ҫзҪ®дёәtrueпјҢдјҡеўһеҠ CPUиҙҹиҪҪпјҡ

жү“ејҖжүӢжңәзҡ„зҪ‘з»ңйҖүжӢ©з•ҢйқўпјҢе°ҶжүӢжңәжіЁеҶҢиҝӣе…ҘзҪ‘з»ңпјҡ

еҰӮжһңжүӢжңәж— жі•жҗңзҙўеҲ°дҝЎеҸ·пјҢжҲ–иҖ…жҗңеҲ°дәҶж— жі•еҠ е…ҘпјҢе…Ҳз”Ёе…¶д»–SDRжҲ–иҖ…йў‘и°ұд»ӘжЈҖжҹҘдёӢжҳҜеҗҰжңүдҝЎеҸ·еҸ‘еҮәпјҢе…¶ж¬ЎжЈҖжҹҘдҝЎеҸ·жҳҜеҗҰжңүйў‘еҒҸпјҢеӣ дёәжҜҸз§ҚжүӢжңәеҜ№йў‘еҒҸзҡ„е®№еҝҚеәҰжҳҜдёҚдёҖж ·зҡ„гҖӮз¬”иҖ…е®һжөӢдёӯпјҢеҚҺдёәзҡ„е…је®№жҖ§жҳҜжңҖеҘҪзҡ„гҖӮ еҰӮжһңжңүйў‘еҒҸпјҢдҝ®ж”№вҖңue.confвҖқдёӯзҡ„вҖңfreq_offsetвҖқеҸӮж•°пјҢеҰӮеӣҫжүҖзӨәгҖӮжӯЈеёёжғ…еҶөдёӢеҰӮжһңдҪҝз”Ёе®ҳж–№еҺҹзүҲSDRи®ҫеӨҮпјҢиҖҢдёҚжҳҜеұұеҜЁSDRи®ҫеӨҮпјҢеҫҲе°‘еҮәзҺ°иҝҷз§Қжғ…еҶөгҖӮеҸҰеӨ–пјҢдҪҝз”ЁGPSDOжҲ–иҖ…е…¶д»–зЁіе®ҡзҡ„ж—¶й’ҹиҫ“е…ҘпјҢд№ҹиғҪйҒҝе…Қиҝҷз§Қй—®йўҳгҖӮ

еҰӮжһңиҝҳжҳҜж— жі•жҗңзҙўеҲ°дҝЎеҸ·пјҢйӮЈе°ұеҸҜиғҪжҳҜжүӢжңәзҡ„еҹәеёҰй—®йўҳпјҢжҜ”еҰӮеңЁй»ҳи®Өй…ҚзҪ®дёӢпјҢжөӢиҜ•еҸ‘зҺ°е°ҸзұіMIXе°ұж— жі•жҗңзҙўеҲ°пјҢеҸҜд»Ҙдҝ®ж”№дёӢйў‘ж®өпјҢд№ҹе°ұжҳҜвҖңEARFCNвҖқеҸӮж•°пјҢд№ҹеҸҜд»ҘзӣҙжҺҘеңЁsrsenbжү§иЎҢж—¶еҠ еҸӮж•°пјҡ

shell
srsenb ~/.config/srslte/enb.conf --rf.dl_earfcn 1575

еҶҚжөӢиҜ•е°ұиғҪжҗңзҙўеҲ°дәҶпјҢеҰӮеӣҫжүҖзӨәгҖӮ

EARFCNе’Ңйў‘ж®өеҜ№з…§е…ізі»еҰӮдёӢеӣҫжүҖзӨәпјҡ

еҪ“з„¶пјҢжіЁеҶҢиҝӣзҪ‘з»ңеҗҺпјҢиҝҳйңҖиҰҒеңЁз”өи„‘дёҠй…ҚзҪ®natиҪ¬еҸ‘пјҡ

shell
echo 1 > /proc/sys/net/ipv4/ip_forward
iptables -t nat -A POSTROUTING -o eth0 -j MASQUERADE

еңЁжүӢжңәдёҠй…ҚзҪ®APNпјҢд№ҹе°ұжҳҜepc.confдёӯй»ҳи®Өи®ҫзҪ®зҡ„srsapnпјҡ

жүӢжңәдёҠAPNи®ҫзҪ®йҮҢж–°е»әпјҢеҸӘйңҖиҰҒи®ҫзҪ®еҗҚз§°е’ҢAPNеҚіеҸҜпјҡ

з„¶еҗҺжүӢжңәе°ұеҸҜд»Ҙж„үеҝ«ең°дёҠзҪ‘дәҶпјҡ

еҶҚиҝӣиЎҢдёҖж¬ЎжөӢйҖҹпјҡ

еҸ—йҷҗдәҺе®¶йҮҢе®ҪеёҰзҡ„йҖҹеәҰпјҢжҖ»дҪ“иҝҳеҸҜд»ҘгҖӮ

еҗҢж—¶пјҢеңЁйғЁзҪІзҡ„з”өи„‘дёҠеҸҜд»ҘдҪҝз”ЁwiresharkиҝӣиЎҢжҠ“еҢ…пјҢеҸҜд»ҘеҜ№жүӢжңәзҡ„4GжөҒйҮҸиҝӣиЎҢеҲҶжһҗпјҢз”ЁдәҺжүӢжңәгҖҒе№іжқҝгҖҒе·ҘжҺ§жңәзӯүдҪҝз”Ё4GзҪ‘з»ңзҡ„и®ҫеӨҮиҝӣиЎҢеҲҶжһҗз ”з©¶гҖӮ

йҷӨдәҶenbз”ЁдәҺжүӢжңәпјҢиҝҳеҸҜд»ҘдҪҝз”Ёueи®©з”өи„‘дёҺз”өи„‘иҝһжҺҘпјҢиҝҷйҮҢдҪҝз”ЁеҸҰдёҖеҸ°з”өи„‘еӨ–жҺҘдёҖдёӘUSRP B210пјҢе®үиЈ…йғЁзҪІжөҒзЁӢдёҖиҮҙпјҢеҸӘжҳҜжү§иЎҢеә”з”Ёзҡ„ж—¶еҖҷдёҚжҳҜsrsepcе’ҢsrsenbпјҢзӣҙжҺҘжү§иЎҢsrsueеҚіеҸҜпјҡ

зӣёеҪ“дәҺиҝҷдёӘUSRPе°ұжҳҜдёҖдёӘLTEеҹәеёҰпјҢиҝһдёҠеҹәз«ҷеҗҺдјҡеҲӣе»әдёҖдёӘиҷҡжӢҹзҪ‘еҚЎtun_srsueпјҢеҲҶй…Қзҡ„IPжҳҜ172.16.0.2е’ҢsrsenbеҒҡеҲӣе»әзҡ„иҷҡжӢҹзҪ‘еҚЎеңЁдёҖдёӘзҪ‘ж®өпјҢеҸҜд»Ҙдә’зӣёpingдёҖдёӢпјҡ

е’Ңsrsenbзұ»дјјsrsueд№ҹеҸҜд»Ҙжү“ејҖжҳҹеә§еӣҫпјҢдҝ®ж”№ue.confпјҢе°Ҷenableи®ҫзҪ®дёәtrueпјҢеҰӮеӣҫжүҖзӨәгҖӮеҗҢж ·дјҡеўһеҠ CPUиҙҹиҪҪпјҢж•ҲжһңеҰӮеӣҫжүҖзӨәгҖӮ

зӣёжҜ”OAIпјҢsrsLTEеҜ№з”өи„‘зҡ„жҖ§иғҪиҰҒжұӮжӣҙй«ҳдёҖдәӣпјҢдҪҶй…ҚзҪ®жӣҙеҠ з®ҖдҫҝгҖӮ

жӣҙеӨҡеҶ…е®№еҸҜд»ҘзңӢе®ҳж–№ж–ҮжЎЈпјҡ https://docs.srslte.com/en/latest/index.html
