Debian安装BCM94352驱动

2016-06-10 | Linux | 阅读
条评论

现在AC盛行，用bgn的速度太慢的完全受不了，所以买的笔记本或者台式都会安一个AC的无线网卡，现在比较常见的有BCM94352和Intel 7260AC，我对比测过，BCM94352的性能比7260AC强一点点，所以我用的网卡基本都是BCM的,Win下还好，Linux下怎么处理？

Ubuntu下如果用图形界面，在设置的驱动支持里面，可以看到网卡驱动，可以直接安装，如果是字符终端呢？

一般只要不用图形界面，我是不会去浪费这个内存和硬盘的，虽然现在也不值钱，但浪费是可耻的~~

不要试图去编译源码，在新的系统上支持很烂，各种编译错误。直接apt安装就行：

apt-get install broadcom-sta-dkms
modprobe -r b44 b43 b43legacy ssb brcmsmac
modprobe wl

﻿安装完成后还需要禁用内核自带的驱动模块：

vi /etc/modprobe.d/broadcom-sta-dkms.conf

做完之后reboot就可以了~~
