华为荣耀X2 GEM-703L B211 ROOT方法

2016-09-19 | Android | 阅读
条评论

为了方便看PDF，买了个华为的荣耀X2 7寸平板手机，财富版和双4G版本一样，只不过内置了股票软件，直接刷双4G固件就可以了。

手机到手后，发现自带贴膜和原装皮套，被卖家忽悠又买了一组，好浪费……

解锁Bootloader

首先要解锁Bootloader，华为很danteng，必须要用登陆云账号14天，过了无条件换货的时间，才能解锁，然后网上的所谓登陆英文网站解锁的方法，被华为很wuyu的做了手脚：如果用华为账号，直接600错误，清cookie后用Facebook账号，验证码无法刷出！不管是国内还是挂国外代理，太keng了！当然，主要原因在于，淘宝花10块，给卖家SN和IMSI，几分钟就能拿到解锁码解锁，呵呵，内外勾结坑大众~~

卡刷B112 OTA升B211

财富版最新系统只到B109，下载双4G版本B112固件，放到SD卡或者手机中，直接本地升级，升级完成后，就是双4G手机了，会收到B211最新固件的推送。

B211 ROOT

通过OTA升级到最新B211之后，刷入当下最新的3.0.2 TWRP后，先备份系统！以防万一，因为系统挂了后，要先刷回官方Recovery，再三键卡刷一个中间版本降级，然后再刷回B112，再OTA升级回到B211，要花费很长时间！我最开始就是被这么坑了，折腾好几个小时才刷回去T_T

很多人Root失败之后，卡在启动界面，就要这么解决。

在网上找Root工具，目前没有支持华为荣耀X2 B211版本的，毕竟Android 6.0。第三方康大的HRT固件自带Root，但毕竟第三方，功能虽然多可稳定性不行啊，还有传闻会偷跑流量！所以，还是踏踏实实的官方固件比较靠谱。

淘宝有人卖Root工具，网上也有一些人卖Root方法，价格都不匪，经过测试找到了一个能够Root的方法，在这里直接给需要Root的人。

下载SuperSU Pro 2.76或者最新的2.78版本（两者的争议可以上网搜），2个版本我都测试过，都可以Root成功，不过出于个人考虑，我选择Chainfire发布的SuperSU Pro 2.76版本，也就是尘大的修改版。

XXXX-SuperSU-vx.xx-xxxx_Mod_by_kane.zip 为未修改刷机包强制模式的版本。
XXXX-SuperSU-vx.xx-xxxx-force-xbin_by_Kane.zip 为修改刷机包脚本为强制 xbin 模式的版本。
XXXX-SuperSU-vx.xx-xxxx-force-system-less.zip 为修改刷机包脚本为强制 system-less 模式的版本。

SuperSU新的版本包含3种，可以看看注意事项和常见提醒：

注意事项及常见提醒：

  • 最新版本提供了 system-less（即不将 supersu 的二进制文件安装至系统，而是对内核打补丁，加载 /data/su.img 闪存盘并执行 root 组件）模式，假如使用 TWRP 刷完 ZIP 包以后，重启或关机提示是否安装 SuperSU，千万不要点安装！！会坏事的！

  • 之前已经通过修改内核来实现 Root 的，请刷回官方原版固件，然后再使用 TWRP 刷入 SuperSU 刷机包！

  • 不支持 Moto 机型 Android 6.0.0 通过直接刷入原版刷机包的方式来 Root 系统，否则会导致无法开机，请使用强制 system-less 的刷机包。之前或之后的 Android 版本无问题（Android 6.0.1 默认使用 system-less 模式，Android 5.x 4.x 默认使用 xbin 模式）。

  • 由于使用 system-less 模式后，会将 root 相关组件放到 /data/ 分区，所以当您重置手机（恢复出厂设置）后，请重新 Root！而使用 xbin 模式重置手机后仍然有 root 组件。

  • 附件中，已添加强制 xbin 模式和强制 system-less 模式的两个 Recovery 刷机包，大家按需选择即可。

如果直接刷以前经常用的xbin模式，刷完之后手机就卡在启动界面无法启动！

如果只刷system-less，启动后会发现没有SuperSU图标，安装apk也会提示没有Root成功。

正确的Root方法是，先刷UPDATE-SuperSU-v2.76-20160705-force-system-less_by_kane.zip，再刷UPDATE-SuperSU-v2.76-20160705-force-xbin_by_kane.zip，启动后，就可以看到SuperSU的图标了，打开也会看到Root成功的提示了！

最后，ROOT后，就别再做OTA升级了，如果要升级最好还是先恢复系统，刷回Recovery，再升级，之后再重新Root，因为谁也不知道售后会不会再其中整什么幺蛾子，如果检测Root直接给变砖了，你也没辙，还不给你售后，只能掏钱维修，狠一点就直接让你换主板，呵呵哒
