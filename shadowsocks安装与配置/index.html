条评论

我的VPS选的ub/debian系统

apt-get install python python-pip

pip install shadowsocks

vi /etc/shadowsocks.json

保存配置文件后启动

开始：ssserver -c /etc/shadowsocks.json -d start
结束：ssserver -c /etc/shadowsocks.json -d stop
